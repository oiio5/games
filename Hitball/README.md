#  Ping-Pong-PyTorch (智能乒乓)

这是一个用 Python 编写的极客小项目，将经典的 **PingPong 游戏** 与现代 **Deep Learning** 结合。
左边的挡板由人类控制，而右边的挡板是由一个真正的人机`PyTorch 神经网络 (3层 MLP)` 在后台进行实时推理和控制的

我为什么做这个项目？因为我觉得在游戏里写一堆 `if-else` 来控制电脑对手太无聊了，不如搭一个神经网络，让它自己学会根据物理规律来预测并追踪球的位置，尝试人机对战。

##  游戏玩法

- **W 键 / S 键** 或 **↑ / ↓ 方向键**：控制左侧玩家挡板上下移动
- **P 键**：暂停 / 继续游戏
- **R 键**：重新开始游戏
- **ESC 键**：直接退出
- **获胜条件**：先获得 **5 分** 的一方获胜！

##  安装步骤

 **安装依赖**：
 我是在vscode中开发运行的
    1. 确保电脑有 Python 3.7+
    2. 解压后进入文件夹，打开终端
    3. 创建并激活虚拟环境（强烈推荐，避免污染全局）
    python -m venv venv *创建虚拟环境*
    venv\Scripts\activate *windows激活虚拟环境*
    pip install -r requirements.txt *在虚拟环境中安装依赖*

 **启动游戏**：
    终端输入 python main.py

## 遇到的问题

**1.输入法焦点劫持：**按下键盘按键没反应，“键盘失灵”
    当键盘输入处于中文输入的时候，就算按下shift切换到英文输入，但只要你还在使用“微软拼音”这类中文输入法，当你按下字母键时，输入法底层依然会去尝试捕获这个按键。这会导致一个致命问题：游戏窗口瞬间失去了焦点
*解决办法：*
    1.彻底切换到纯英文键盘：在Windows电脑上，按下 Win 键 + 空格键，把输入法彻底切换到英文。
    2.点一下游戏窗口：用鼠标在黑色的游戏画面上随便点一下，把焦点还给游戏
    3.在游戏初始化时，直接用Pygame的内置指令把输入法屏蔽掉：即在pygame.init() 后添加pygame.key.stop_text_input()
    *但是在屏蔽输入法后，虽然输入法处于中文键入状态是系统可以识别到，但是Windows自带的微软拼音还会强制置顶输入框的UI*

**2.中文显示乱码**
    导入os库，修改get_chinese_font() 函数，直接读取系统底层的中文字体文件
